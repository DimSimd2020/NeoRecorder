Проект: "NeoRecorder" (Рабочее название)
1. Концепция и Философия

Суть: Максимально легковесный, портативный (Portable) инструмент для записи экрана без установки, рекламы и водяных знаков.
Vibe (Стиль): Киберпанк в тёмно сером основном цвете и дополнительным синим неоновым цветом. Никаких стандартных серых окон Windows 95. Интерфейс должен выглядеть как приборная панель современного гаджета.
2. Техническое Задание (ТЗ)
2.1. Функциональные требования

    Режимы захвата:

        Fullscreen: Весь экран (с поддержкой мультимониторности).

        Region: Выделение прямоугольной области мышью (с затемнением остального экрана).

        Window: Автоматическое определение открытых окон, подсветка при наведении, захват только содержимого окна (даже если его перекрывают другие).

    Аудио:

        Запись системных звуков (Loopback).

        Запись микрофона.

        Смешанный режим (Система + Микрофон).

        Визуализация уровня звука (VU Meter) в интерфейсе, чтобы видеть, что микрофон работает.

    Управление:

        Горячие клавиши (Start/Stop/Pause).

        Сворачивание в трей при записи.

        Плавающий виджет управления (опционально).

    Вывод:

        Формат: .mp4 (H.264).

        Автоматическое именование файлов (Дата_Время).

2.2. Стек технологий (Architecture Stack)

    Язык: Python 3.10+.

    GUI Фреймворк: CustomTkinter (CTk). Это обертка над Tkinter, которая дает современные скругленные углы, темную тему из коробки и выглядит нативно в Win11.

    Core (Запись):

        Вариант А (Хардкорный): FFmpeg (через subprocess) — для идеальной синхронизации и производительности.

        Вариант Б (Нативный): MSS (видео) + PyAudio (звук) + MoviePy (сборка). Рекомендую Вариант А для продакшна, Б для учебного проекта.

    Сборка: PyInstaller (флаг --onefile).

3. UI/UX Дизайн

Представим интерфейс. Это должно быть компактное окно, не перегруженное кнопками.
Цветовая палитра (Dark Theme)

    Фон: #2B2B2B (Глубокий серый).

    Акцент (Кнопки): #1F6AA5 (Спокойный синий).

    Текст: #FFFFFF и #A0A0A0 (для второстепенного).

Макет (Wireframe)

Окно разделено на 3 логических блока сверху вниз:

    Блок "Источник видео" (Верх):

        Три большие иконки-кнопки (Toggle buttons): [Экран], [Область], [Окно].

        При выборе [Окно] появляется выпадающий список с названиями активных приложений.

    Блок "Источник звука" (Середина):

        Чекбоксы или переключатели: [Микрофон], [Система].

        Рядом с микрофоном — выпадающий список устройств (если их несколько).

        Фича: Полоска-индикатор громкости, которая дергается, когда ты говоришь (сразу видно, работает ли микрофон).

    Блок "Действие" (Низ):

        Огромная круглая кнопка REC.

        Рядом маленькая кнопка "Настройки" (шестеренка) и "Папка с записями".

        Таймер записи (появляется, когда запись идет): 00:00:00.    
Локализация (Localization)

    Поддержка английского и русского языков и удобное масштабирование для других языков через централизированную систему локализации с помощью lang/json файлов assets/lang
